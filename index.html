Frequently Asked Questions

Common questions about this demonstration application.

Is this a real loan application?

No. This project is a demonstration interface created for portfolio and presentation purposes. No personal information is submitted to any lender.

How is the loan limit determined?

The demo calculates eligibility using employment category and monthly income to generate a realistic pre-approved amount between KSh 5,000 and KSh 18,000.

How long does the application take?

Most users can complete the full application in under five minutes.
